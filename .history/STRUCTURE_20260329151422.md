# 项目结构

```
agentic-rag/
├── .github/
│   └── workflows/
│       └── ci.yml              # GitHub Actions CI 配置
├── agentic_rag/                # 主 Python 包
│   ├── __init__.py             # 包初始化
│   ├── config.py               # 全局配置
│   ├── main.py                 # 入口脚本
│   ├── api/                    # FastAPI 后端
│   │   └── __init__.py
│   ├── core/                   # 核心功能模块
│   │   ├── __init__.py         # 文档处理、ES检索、LLM封装
│   │   ├── agent.py            # Agentic RAG 工作流
│   │   └── cache.py            # 对话缓存
│   ├── db/                     # 数据库层
│   │   └── __init__.py         # SQLAlchemy 模型和 CRUD
│   └── frontend/               # 前端文件
│       ├── __init__.py
│       ├── api.js              # JS API 客户端
│       ├── app.js              # JS 前端逻辑
│       ├── index.html          # HTML 主页面
│       ├── style.css           # 样式文件
│       └── streamlit_app.py    # Streamlit 前端
├── docs/                       # 文档
│   ├── architecture.md         # 架构文档
│   ├── api.md                  # API 文档
│   └── deployment.md           # 部署指南
├── tests/                      # 测试代码
│   └── test_basic.py
├── .env.example                # 环境变量模板
├── .gitignore                  # Git 忽略文件
├── LICENSE                     # MIT 许可证
├── README.md                   # 项目说明
└── requirements.txt            # Python 依赖
```

## 快速启动

```bash
# 安装
pip install -r requirements.txt

# 配置
cp .env.example .env
# 编辑 .env 文件

# 启动
python -m agentic_rag.main
```

## 开发

```bash
# 代码格式化
black agentic_rag/
isort agentic_rag/

# 测试
pytest tests/

# 类型检查
mypy agentic_rag/
```
