│  └──────┬───────┘  └──────┬───────┘  └──────┬───────┘      │
└─────────┼─────────────────┼─────────────────┼──────────────┘

          └─────────────────┴─────────────────┘

┌───────────────────────────▼─────────────────────────────────┐

## 检索流程

1. **混合检索**：BM25 + 向量相似度
2. **候选池**：从 ES 获取 top 20 结果
3. **重排序**：使用 CrossEncoder 重排序
4. **最终选择**：取 top 5 作为上下文
